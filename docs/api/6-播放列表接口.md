# 播放列表接口

> 文档地图：[README](../../README.md) > [API接口文档](../../README.md#api-接口文档) > 本文档

> 认证：除标注「公开」的接口外，需要请求头携带 `token`。无效 token 返回 HTTP 403。
>
> 详细业务流程见对应的 [业务文档](../业务/9-播放列表.md)。

## 接口列表

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 创建播放列表 | POST | /playlist/createPlaylist | 需要 |
| 更新播放列表 | POST | /playlist/updatePlaylist | 需要 |
| 删除播放列表 | GET | /playlist/deletePlaylist | 需要 |
| 恢复播放列表 | GET | /playlist/recoverPlaylist | 需要 |
| 获取播放列表详情 | GET | /playlist/getPlaylistById | 需要 |
| 获取播放列表视频详情 | GET | /playlist/getPlayItemListDetail | 需要 |
| 获取用户的播放列表 | GET | /playlist/getPlaylistByPage | 不需要 |
| 获取我的播放列表 | GET | /playlist/getMyPlaylistByPage | 需要 |
| 添加视频到播放列表 | POST | /playlist/addPlaylistItem | 需要 |
| 删除播放列表中的视频 | POST | /playlist/deletePlaylistItem | 需要 |
| 移动视频位置 | POST | /playlist/movePlaylistItem | 需要 |
| 获取包含该视频的播放列表 | GET | /playlist/getPlaylistByVideoId | 需要 |

## 接口详情

### 创建播放列表

**请求**
```
POST /playlist/createPlaylist
Header: token: {your_token}
Content-Type: application/json
```

**请求体**
```json
{
    "title": "我的播放列表",
    "description": "播放列表描述"
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | String | 是 | 播放列表标题 |
| description | String | 否 | 描述 |

---

### 更新播放列表

**请求**
```
POST /playlist/updatePlaylist
Header: token: {your_token}
Content-Type: application/json
```

**请求体**
```json
{
    "playlistId": "pl_123456",
    "title": "新标题",
    "description": "新描述",
    "visibility": "PUBLIC"
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |
| title | String | 否 | 标题 |
| description | String | 否 | 描述 |
| visibility | String | 否 | 可见性：PUBLIC / PRIVATE |

---

### 删除播放列表

逻辑删除，可通过recoverPlaylist恢复。

**请求**
```
GET /playlist/deletePlaylist?playlistId={playlistId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |

---

### 恢复播放列表

恢复已逻辑删除的播放列表。

**请求**
```
GET /playlist/recoverPlaylist?playlistId={playlistId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |

---

### 获取播放列表详情

**请求**
```
GET /playlist/getPlaylistById?playlistId={playlistId}&showVideoList=false
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |
| showVideoList | Boolean | 否 | 是否返回视频列表，默认false |

---

### 获取播放列表视频详情

获取播放列表中所有视频的详细信息。

**请求**
```
GET /playlist/getPlayItemListDetail?playlistId={playlistId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |

---

### 获取用户的播放列表

分页获取指定用户的播放列表。

**请求**
```
GET /playlist/getPlaylistByPage?userId={userId}&skip=0&limit=10
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | String | 是 | 用户ID |
| skip | int | 是 | 跳过条数 |
| limit | int | 是 | 返回条数 |

---

### 获取我的播放列表 🔒

分页获取当前登录用户的播放列表。

**请求**
```
GET /playlist/getMyPlaylistByPage?skip=0&limit=10
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| skip | int | 是 | 跳过条数 |
| limit | int | 是 | 返回条数 |

**响应示例**
```json
{
    "code": 0,
    "data": [],
    "message": "成功"
}
```

---

### 添加视频到播放列表 🔒

**请求**
```
POST /playlist/addPlaylistItem
Header: token: {your_token}
Content-Type: application/json
```

**请求体**
```json
{
    "playlistId": "pl_123456",
    "videoIdList": ["v_001", "v_002"],
    "addMode": "APPEND"
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |
| videoIdList | List\<String\> | 是 | 要添加的视频ID列表 |
| addMode | String | 否 | 添加模式：APPEND 等 |

---

### 删除播放列表中的视频

**请求**
```
POST /playlist/deletePlaylistItem
Header: token: {your_token}
Content-Type: application/json
```

**请求体**
```json
{
    "playlistId": "pl_123456",
    "deleteMode": "BY_VIDEO_ID",
    "videoIdList": ["v_001"]
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |
| deleteMode | String | 是 | 删除模式 |
| videoIdList | List\<String\> | 否 | 按视频ID删除 |
| playItemIdList | List\<String\> | 否 | 按播放项ID删除 |
| singleIndex | Integer | 否 | 按单个索引删除 |
| indexList | List\<Integer\> | 否 | 按索引列表删除 |
| startIndex | Integer | 否 | 范围删除起始索引 |
| endIndex | Integer | 否 | 范围删除结束索引 |
| startTime | Date | 否 | 按时间范围删除起始 |
| endTime | Date | 否 | 按时间范围删除结束 |

---

### 移动视频位置

调整视频在播放列表中的排列顺序。

**请求**
```
POST /playlist/movePlaylistItem
Header: token: {your_token}
Content-Type: application/json
```

**请求体**
```json
{
    "playlistId": "pl_123456",
    "videoId": "v_001",
    "moveMode": "TO_INDEX",
    "toIndex": 0
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| playlistId | String | 是 | 播放列表ID |
| videoId | String | 是 | 要移动的视频ID |
| moveMode | String | 是 | 移动模式：TO_INDEX / TO_VIDEO 等 |
| toIndex | Integer | 否 | 目标索引位置 |
| toVideoId | String | 否 | 移动到指定视频之前 |

---

### 获取包含该视频的播放列表

反向查询：获取包含指定视频的所有播放列表。

**请求**
```
GET /playlist/getPlaylistByVideoId?videoId={videoId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |
