# 点赞接口

> 文档地图：[README](../../README.md) > [API接口文档](../../README.md#api-接口文档) > 本文档

> 认证：除标注「公开」的接口外，需要请求头携带 `token`。无效 token 返回 HTTP 403。
>
> 详细业务流程见对应的 [业务文档](../业务/)。

## 接口列表

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 视频点赞 | GET | /videoLike/like | 需要 |
| 视频点踩 | GET | /videoLike/dislike | 需要 |
| 获取点赞状态 | GET | /videoLike/getStatus | 不需要 |

## 接口详情

### 视频点赞 🔒

Toggle 方式：已点赞则取消，已点踩则切换为点赞。

**请求**
```
GET /videoLike/like?videoId={videoId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": null
}
```

### 获取点赞状态

获取视频的点赞/点踩数量及当前用户的操作状态。

**请求**
```
GET /videoLike/getStatus?videoId={videoId}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |

**响应示例（已登录用户）**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "likeCount": 15,
        "dislikeCount": 2,
        "userAction": "LIKE"
    }
}
```

**响应示例（未登录用户）**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "likeCount": 15,
        "dislikeCount": 2,
        "userAction": null
    }
}
```

| 字段 | 说明 |
|------|------|
| likeCount | 点赞总数 |
| dislikeCount | 点踩总数 |
| userAction | 当前用户的操作：LIKE / DISLIKE / null（未操作） |
