# 播放视频接口

> 文档地图：[README](../../README.md) > [API接口文档](../../README.md#api-接口文档) > 本文档

## 接口列表

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 获取播放信息 | GET | /watchController/getWatchInfo | 需clientId或token |
| 自适应码率主播放列表 | GET | /watchController/getMultivariantPlaylist.m3u8 | 不需要 |
| 单分辨率TS列表 | GET | /watchController/getM3u8Content.m3u8 | 不需要 |
| 记录播放日志 | GET | /watchController/addWatchLog | 不需要 |
| 心跳上报 | POST | /heartbeat/add | 不需要 |
| 获取播放进度 | GET | /progress/getProgress | 不需要 |
| 文件访问（带签名） | GET | /file/access | 不需要 |

## 接口详情

### 获取播放信息

根据watchId获取视频播放所需信息，包括封面、状态、播放列表URL等。

**请求**
```
GET /watchController/getWatchInfo?watchId={watchId}
Header: token: {your_token}（或通过clientId识别用户）
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| watchId | String | 是 | 观看ID |

**响应示例**
```json
{
    "code": 0,
    "data": {
        "videoId": "v_2507678200001472",
        "videoStatus": "TRANSCODING",
        "coverUrl": "https://video-2022-dev.oss-cn-beijing.aliyuncs.com/...",
        "multivariantPlaylistUrl": "http://localhost:5022/watchController/getMultivariantPlaylist.m3u8?videoId=v_2507678200001472&clientId=...&sessionId=...",
        "progressInMillis": 0
    },
    "message": "成功"
}
```

| 字段 | 说明 |
|------|------|
| videoId | 视频ID |
| videoStatus | 视频状态：READY / TRANSCODING 等 |
| coverUrl | 封面图URL |
| multivariantPlaylistUrl | HLS自适应码率主播放列表地址 |
| progressInMillis | 上次播放进度（毫秒） |

---

### 自适应码率主播放列表

返回HLS多码率主播放列表（m3u8），播放器根据网络状况自动选择分辨率。

**请求**
```
GET /watchController/getMultivariantPlaylist.m3u8?videoId={videoId}&clientId={clientId}&sessionId={sessionId}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |
| clientId | String | 否 | 客户端ID |
| sessionId | String | 否 | 会话ID |

**响应示例**（text/application/vnd.apple.mpegurl）
```
#EXTM3U

#EXT-X-STREAM-INF:BANDWIDTH=7032284,AVERAGE-BANDWIDTH=4064092
/watchController/getM3u8Content.m3u8?resolution=720p&videoId=...&transcodeId=...

#EXT-X-STREAM-INF:BANDWIDTH=18344529,AVERAGE-BANDWIDTH=11883132
/watchController/getM3u8Content.m3u8?resolution=1080p&videoId=...&transcodeId=...
```

---

### 单分辨率TS列表

返回指定分辨率的TS切片列表。

**请求**
```
GET /watchController/getM3u8Content.m3u8?transcodeId={transcodeId}&resolution={resolution}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| transcodeId | String | 是 | 转码任务ID |
| resolution | String | 是 | 分辨率：720p / 1080p 等 |

**响应示例**（text/application/vnd.apple.mpegurl）
```
#EXTM3U
#EXT-X-VERSION:3
#EXT-X-ALLOW-CACHE:YES
#EXT-X-TARGETDURATION:2
#EXT-X-MEDIA-SEQUENCE:0
#EXTINF:1.983333,
/file/access?resolution=720p&tsIndex=0&fileId=...&timestamp=...&nonce=...&sign=...
#EXTINF:1.983333,
/file/access?resolution=720p&tsIndex=1&fileId=...&timestamp=...&nonce=...&sign=...
#EXT-X-ENDLIST
```

### 记录播放日志

记录用户播放行为日志。

**请求**
```
GET /watchController/addWatchLog?videoStatus={videoStatus}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoStatus | String | 是 | 视频状态 |

---

### 心跳上报

播放器定时上报播放状态、进度等信息。

**请求**
```
POST /heartbeat/add
Content-Type: application/json
```

**请求体**
```json
{
    "videoId": "v_2507678200001472",
    "clientId": "69aaf146e0df6537cc235392",
    "sessionId": "69aaf146e0df6537cc235391",
    "videoStatus": "READY",
    "playerProvider": "ALIYUN_WEB",
    "clientTime": "2022-12-20T16:11:47.181Z",
    "type": "TIMER",
    "event": null,
    "playerTime": 820,
    "playerStatus": "pause",
    "playerVolume": 1
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |
| clientId | String | 是 | 客户端ID |
| sessionId | String | 是 | 会话ID |
| videoStatus | String | 否 | 视频状态 |
| playerProvider | String | 否 | 播放器类型 |
| clientTime | String | 否 | 客户端时间（ISO 8601） |
| type | String | 否 | 心跳类型：TIMER / EVENT |
| event | String | 否 | 事件名称 |
| playerTime | Long | 否 | 播放进度（秒） |
| playerStatus | String | 否 | 播放状态：playing / pause |
| playerVolume | BigDecimal | 否 | 音量（0-1） |

### 获取播放进度

获取用户对某视频的播放进度。

**请求**
```
GET /progress/getProgress?videoId={videoId}&clientId={clientId}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |
| clientId | String | 是 | 客户端ID |

**响应示例**
```json
{
    "code": 0,
    "data": null,
    "message": "成功"
}
```

> data为null表示无播放记录，有记录时返回进度信息。

---

### 文件访问（带签名）

通过签名验证访问视频TS切片文件，由m3u8列表中的URL自动生成。

**请求**
```
GET /file/access?resolution={resolution}&fileId={fileId}&timestamp={timestamp}&nonce={nonce}&sign={sign}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| resolution | String | 是 | 分辨率 |
| fileId | String | 是 | 文件ID |
| timestamp | String | 是 | 时间戳 |
| nonce | String | 是 | 随机字符串 |
| sign | String | 是 | 签名（由后端生成） |

> 此接口返回302重定向到OSS文件地址，签名参数由getM3u8Content接口自动生成。
