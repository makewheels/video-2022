## 接口详情

### 创建视频

创建视频记录，返回videoId和fileId，用于后续上传。

**请求**
```
POST /video/create
Header: token: {your_token}
Content-Type: application/json
```

**请求体**
```json
{
    "rawFilename": "VID_20220319_131135.mp4",
    "videoType": "USER_UPLOAD",
    "size": 104857600,
    "ttl": "PERMANENT"
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| rawFilename | String | 是 | 原始文件名，用于确定上传到对象存储的文件后缀 |
| videoType | String | 是 | 类型：USER_UPLOAD / YOUTUBE |
| size | Long | 否 | 文件大小（字节） |
| ttl | String | 否 | 有效期：PERMANENT（永久）等 |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "watchId": "2ISMYP",
        "shortUrl": "http://localhost:5022/w?v=2ISMYP",
        "videoId": "v_2507678200001472",
        "watchUrl": "http://localhost:5022/w?v=2ISMYP",
        "fileId": "f_2507678200001210"
    }
}
```

| 字段 | 说明 |
|------|------|
| videoId | 视频ID，后续接口使用 |
| fileId | 文件ID，用于上传对象存储 |
| watchId | 观看ID |
| watchUrl | 观看地址 |
| shortUrl | 短链接 |

### 获取上传凭证

通过阿里云OSS STS接口，获取临时上传凭证。

**请求**
```
GET /file/getUploadCredentials?fileId={fileId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| fileId | String | 是 | 文件ID（创建视频时返回） |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "bucket": "video-2022-dev",
        "accessKeyId": "STS.NURKKUmgubZ1ugs1fZZqrTqLj",
        "endpoint": "oss-cn-beijing.aliyuncs.com",
        "secretKey": "3zdCPTmzEhVq7CMy1kpRjpzhESRPpMD5viNADtB3C3bq",
        "provider": "ALIYUN_OSS",
        "sessionToken": "CAISogJ1q6Ft5B2yfSjIr5bnAPHh...",
        "expiration": "2022-12-28T13:16:37Z",
        "key": "videos/{userId}/{videoId}/raw/{fileId}.mp4"
    }
}
```

| 字段 | 说明 |
|------|------|
| bucket | OSS存储桶 |
| accessKeyId | 临时AccessKey ID |
| secretKey | 临时AccessKey Secret |
| endpoint | OSS公网地址 |
| provider | 存储供应商（ALIYUN_OSS） |
| sessionToken | STS临时Token |
| expiration | 凭证过期时间 |
| key | 上传到OSS的对象路径 |


| showWatchCount | Boolean | 否 | 是否展示观看次数 |

### 获取视频详情

**请求**
```
GET /video/getVideoDetail?videoId={videoId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |

**响应示例**
```json
{
    "code": 0,
    "data": {
        "id": "v_2507678200001472",
        "userId": "u_2507677900001740",
        "type": "USER_UPLOAD",
        "status": "TRANSCODING",
        "title": null,
        "description": null,
        "duration": 3000,
        "watchCount": 0,
        "watchId": "2ISMYP",
        "watchUrl": "http://localhost:5022/w?v=2ISMYP",
        "shortUrl": "http://localhost:5022/w?v=2ISMYP",
        "coverUrl": "https://video-2022-dev.oss-cn-beijing.aliyuncs.com/...",
        "createTime": "2026-03-06T15:05:29.406Z",
        "createTimeString": "2026-03-06 23:05:29",
        "youtubePublishTime": null,
        "youtubePublishTimeString": null
    },
    "message": "成功"
}
```

| 字段 | 说明 |
|------|------|
| status | 视频状态：TRANSCODING（转码中）/ READY（就绪）等 |
| duration | 视频时长（毫秒） |
| watchCount | 观看次数 |
| coverUrl | 封面图URL（带签名） |

### 获取我的视频列表

分页获取当前用户的视频列表。

**请求**
```
GET /video/getMyVideoList?skip=0&limit=10
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| skip | int | 是 | 跳过条数 |
| limit | int | 是 | 返回条数 |

**响应示例**
```json
{
    "code": 0,
    "data": [
        {
            "id": "v_2507678200001472",
            "status": "TRANSCODING",
            "title": null,
            "description": null,
            "duration": null,

            "createTimeString": "2026-03-06 23:05:29"

```
