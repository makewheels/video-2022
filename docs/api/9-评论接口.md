# 评论接口

> 文档地图：[README](../../README.md) > [API接口文档](../../README.md#api-接口文档) > 本文档

> 认证：除标注「公开」的接口外，需要请求头携带 `token`。无效 token 返回 HTTP 403。
>
> 详细业务流程见对应的 [业务文档](../业务/)。

## 接口列表

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 添加评论 | POST | /comment/add | 需要 |
| 获取视频评论 | GET | /comment/getByVideoId | 不需要 |
| 获取回复列表 | GET | /comment/getReplies | 不需要 |
| 删除评论 | GET | /comment/delete | 需要 |
| 评论点赞 | GET | /comment/like | 需要 |
| 获取评论数 | GET | /comment/getCount | 不需要 |

## 接口详情

### 添加评论 🔒

添加顶级评论或回复。回复子评论时，parentId 自动指向顶级评论。

**请求**
```
POST /comment/add
Header: token: {your_token}
Content-Type: application/json
```

**请求体**
```json
{
    "videoId": "v_2507678200001472",
    "content": "很棒的视频！",
    "parentId": null
}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |
| content | String | 是 | 评论内容 |
| parentId | String | 否 | 父评论ID，null 表示顶级评论 |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "id": "69ac3026d01ba9a13a565700",
        "videoId": "v_2507678200001472",
        "userId": "u_2507677900001740",
        "userPhone": "188****5678",
        "content": "很棒的视频！",
        "parentId": null,
        "replyToUserId": null,
        "replyToUserPhone": null,
        "likeCount": 0,
        "replyCount": 0,
        "createTime": "2026-03-07T10:30:00.000Z",
        "updateTime": "2026-03-07T10:30:00.000Z"
    }
}
```

| 字段 | 说明 |
|------|------|
| userPhone | 脱敏手机号（如 188****5678） |
| parentId | 父评论ID，顶级评论为 null |
| replyToUserId | 被回复的用户ID（仅回复时有值） |
| likeCount | 点赞数 |
| replyCount | 回复数（仅顶级评论） |

### 获取视频评论

分页获取视频的顶级评论，支持按时间或热度排序。

**请求**
```
GET /comment/getByVideoId?videoId={videoId}&skip=0&limit=20&sort=time
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |
| skip | int | 否 | 跳过条数（默认 0） |
| limit | int | 否 | 返回条数（默认 20） |
| sort | String | 否 | 排序方式：time（按时间倒序）/ hot（按点赞数倒序），默认 time |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": [
        {
            "id": "69ac3026d01ba9a13a565700",
            "videoId": "v_2507678200001472",
            "userId": "u_2507677900001740",
            "userPhone": "188****5678",
            "content": "很棒的视频！",
            "parentId": null,
            "likeCount": 5,
            "replyCount": 2,
            "createTime": "2026-03-07T10:30:00.000Z"
        }
    ]
}
```

### 获取回复列表

分页获取某条顶级评论的回复，按时间升序排列。

**请求**
```
GET /comment/getReplies?parentId={parentId}&skip=0&limit=20
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| parentId | String | 是 | 父评论ID |
| skip | int | 否 | 跳过条数（默认 0） |
| limit | int | 否 | 返回条数（默认 20） |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": [
        {
            "id": "69ac3027d01ba9a13a565701",
            "videoId": "v_2507678200001472",
            "userId": "u_2507677900001741",
            "userPhone": "139****6789",
            "content": "同意！",
            "parentId": "69ac3026d01ba9a13a565700",
            "replyToUserId": "u_2507677900001740",
            "replyToUserPhone": "188****5678",
            "likeCount": 0,
            "replyCount": 0,
            "createTime": "2026-03-07T10:31:00.000Z"
        }
    ]
}
```

### 删除评论 🔒

删除评论。仅评论作者或视频上传者可操作。删除顶级评论时级联删除所有回复及点赞记录。

**请求**
```
GET /comment/delete?commentId={commentId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| commentId | String | 是 | 评论ID |

**响应示例（成功）**
```json
{
    "code": 0,
    "message": "成功",
    "data": null
}
```

**响应示例（无权限）**
```json
{
    "code": 1,
    "message": "无权删除此评论",
    "data": null
}
```

### 评论点赞 🔒

Toggle 方式点赞：已点赞则取消，未点赞则添加。

**请求**
```
GET /comment/like?commentId={commentId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| commentId | String | 是 | 评论ID |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": null
}
```

### 获取评论数

获取视频的总评论数（包含回复）。

**请求**
```
GET /comment/getCount?videoId={videoId}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "count": 42
    }
}
```
