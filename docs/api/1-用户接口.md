# 用户接口

> 文档地图：[README](../../README.md) > [API接口文档](../../README.md#api-接口文档) > 本文档

## 接口列表

| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 请求验证码 | GET | /user/requestVerificationCode | 不需要 |
| 提交验证码 | GET | /user/submitVerificationCode | 不需要 |
| 通过token获取用户 | GET | /user/getUserByToken | 不需要 |
| 通过ID获取用户 | GET | /user/getUserById | 不需要 |
| 获取客户端ID | GET | /client/requestClientId | 不需要 |
| 获取会话ID | GET | /session/requestSessionId | 不需要 |

## 接口详情

### 请求验证码

请求短信验证码，后端会在Redis中缓存，有过期时间。

**请求**
```
GET /user/requestVerificationCode?phone={phone}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| phone | String | 是 | 手机号 |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": null
}
```

---

### 提交验证码

验证短信验证码，验证成功返回用户信息和token。

**请求**
```
GET /user/submitVerificationCode?phone={phone}&code={code}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| phone | String | 是 | 手机号 |
| code | String | 是 | 验证码 |

**响应示例（验证成功）**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "id": "u_2507677900001740",
        "phone": "18812345678",
        "registerChannel": "WEB",
        "createTime": "2026-03-06T15:05:19.187Z",
        "token": "2029936368066678784"
    }
}
```

**响应示例（验证失败）**
```json
{
    "code": 1,
    "message": "fail",
    "data": null
}
```

| 字段 | 说明 |
|------|------|
| id | 用户ID |
| phone | 手机号 |
| registerChannel | 注册渠道：WEB / WECHAT_MINI_PROGRAM |
| createTime | 创建时间 |
| token | 登录凭证，用于后续认证接口 |

---

### 通过token获取用户

**请求**
```
GET /user/getUserByToken?token={token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| token | String | 是 | 登录凭证 |

**响应示例**
```json
{
    "code": 0,
    "data": {
        "id": "u_2507677900001740",
        "phone": "18812345678",
        "registerChannel": null,
        "createTime": "2026-03-06T15:05:19.187Z",
        "updateTime": "2026-03-06T15:05:19.187Z",
        "token": "2029936368066678784"
    },
    "message": "成功"
}
```

---

### 通过ID获取用户

**请求**
```
GET /user/getUserById?userId={userId}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | String | 是 | 用户ID |

**响应示例**
```json
{
    "code": 0,
    "data": {
        "id": "u_2507677900001740",
        "phone": "18812345678",
        "registerChannel": null,
        "createTime": "2026-03-06T15:05:19.187Z",
        "updateTime": "2026-03-06T15:05:19.187Z",
        "token": null
    },
    "message": "成功"
}
```

> 注意：通过ID查询时，token字段不返回。

---

### 获取客户端ID

用户无需登录即可观看视频。clientId用于追踪未登录用户，前端保存在localStorage中。

**请求**
```
GET /client/requestClientId
```

**响应示例**
```json
{
    "code": 0,
    "data": {
        "clientId": "69aaf146e0df6537cc235392"
    },
    "message": "成功"
}
```

---

### 获取会话ID

用于区分观看次数，前端保存在sessionStorage中。

**请求**
```
GET /session/requestSessionId
```

**响应示例**
```json
{
    "code": 0,
    "data": {
        "sessionId": "69aaf146e0df6537cc235391"
    },
    "message": "成功"
}
```
