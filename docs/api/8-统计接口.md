| 接口 | 方法 | 路径 | 认证 |
|------|------|------|------|
| 获取视频流量消耗 | GET | /statistics/getTrafficConsume | 需要 |
| 聚合流量统计数据 | GET | /statistics/aggregateTrafficData | 需要 |

## 接口详情

### 获取视频流量消耗

统计某个视频被观众观看所消耗的流量。

**请求**
```
GET /statistics/getTrafficConsume?videoId={videoId}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| videoId | String | 是 | 视频ID |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "trafficConsumeString": "693.02 MB",
        "videoId": "63a1a40bcc99952a50decc3f",
        "trafficConsumeInBytes": 726685236
    }
}
```

| 字段 | 说明 |
|------|------|
| trafficConsumeString | 人类可读的流量消耗 |
| trafficConsumeInBytes | 流量消耗（字节） |
| videoId | 视频ID |

---

### 聚合流量统计数据

按天聚合流量消耗数据，用于前端Echarts图表展示。

**请求**
```
GET /statistics/aggregateTrafficData?startTime={startTime}&endTime={endTime}
Header: token: {your_token}
```

**参数**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| startTime | long | 是 | 开始时间（毫秒时间戳） |
| endTime | long | 是 | 结束时间（毫秒时间戳） |

**响应示例**
```json
{
    "code": 0,
    "message": "成功",
    "data": {
        "xAxis": {
            "data": ["2023-12-05", "2023-12-08", "2023-12-11"],
            "type": "category"
        },
        "yAxis": {
            "data": ["20.65 MB", "139.07 MB", "9.43 MB"],
            "type": "value"
        },
        "series": {
            "data": [21654968, 145822764, 9890492],
            "label": { "show": true, "position": "top" },
            "type": "bar"
        }
    }
}
```

| 字段 | 说明 |
|------|------|
| xAxis.data | 日期列表 |
| yAxis.data | 人类可读的流量值 |
| series.data | 流量值（字节），用于图表绘制 |
```
