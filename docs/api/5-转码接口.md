## 接口详情

### 阿里云转码回调

阿里云云函数ffmpeg转码完成后的回调接口。转码服务在完成视频转码后，调用此接口通知后端更新转码状态。

**请求**
```
POST /transcode/aliyunCloudFunctionTranscodeCallback
Content-Type: application/json
```

**请求体**

由阿里云转码服务发送的JSON回调数据，包含转码任务的执行结果。

**转码流程说明**

1. 后端调用阿里云云函数发起转码任务
2. 云函数使用ffmpeg进行视频转码（切片为HLS TS格式）
3. 转码完成后，云函数回调此接口通知结果
4. 后端更新视频状态为READY

**相关的阿里云云函数接口**（由后端内部调用，非对外接口）

- ffprobe获取视频信息：`POST https://ffprobe-video-transcode-{id}.cn-beijing.fcapp.run`
- 发起转码：`POST https://transcode-master-video-transcode-{id}.cn-beijing.fcapp.run`
