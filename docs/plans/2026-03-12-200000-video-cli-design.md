# video-cli: 视频平台命令行工具设计

## 概述

为 video-2022 视频平台构建 Python Click CLI 工具，遵循 CLI-Anything 模式。让 AI Agent 和脚本能通过命令行管理视频、用户、评论、播放列表等所有平台功能。

## 技术方案

- **语言**: Python 3.10+
- **框架**: Click (命令行) + Requests (HTTP)
- **输出**: JSON (默认) + table (人类可读)
- **安装**: `pip install -e .`
- **命令名**: `video-cli`

## 命令结构

```
video-cli [--base-url URL] [--token TOKEN] [--output json|table]

auth:     login, me
video:    create, list, detail, update, delete, status
comment:  add, list, delete, like, count
like:     like, dislike, status
playlist: create, list, detail, add-item, delete-item, delete
youtube:  info, transfer
stats:    traffic, aggregate
watch:    info, start, heartbeat
```

## 核心特性

1. **JSON 输出** — 所有命令默认输出 JSON，Agent 直接解析
2. **Token 持久化** — 登录后保存到 `~/.video-cli/config.json`
3. **自描述** — `--help` 显示完整命令文档
4. **统一错误** — exit code + JSON 错误消息
5. **可配置** — base-url 支持本地/远程/自定义服务器
