# "我的视频"页面改造设计

## 目标

将首页（index.html）改造为"我的视频"页面，以视频列表为核心，增加分页功能，响应式适配电脑和手机。

## 设计决策

### 导航栏
- "首页" → "我的视频"（所有6个页面 + watch.html 同步改）

### 页面内容
- **去掉**：欢迎区（welcome-section）
- **保留**：快捷操作卡片（上传、统计、YouTube搬运）
- **增强**：视频列表网格

### 视频卡片增强
现有：标题 + 播放次数
新增：
- 创建时间（相对时间，如"3天前"）
- 视频时长（格式化，如"02:30"）
- 视频状态标签（上传中/已完成）

### 分页策略（响应式）

**电脑端（>768px）**：传统分页器
- 每页12条
- 底部分页器："上一页 1 2 3 ... 下一页"
- 显示总数："共 XX 个视频"

**手机端（≤768px）**："加载更多"按钮
- 初始加载12条
- 底部"加载更多"按钮，点击追加12条
- 全部加载完显示"没有更多了"

### 后端改动
- `getMyVideoList` API 返回值改为包含 `total` 总数
- 返回格式：`{ code: 0, data: { list: [...], total: N } }`
