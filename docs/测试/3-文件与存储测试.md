# 文件与存储测试

文件与存储模块覆盖文件生命周期管理（创建、上传完成、删除）、文件访问日志、MD5 校验、OSS 对象存储操作以及路径生成工具。测试确保文件系统与云存储的正确交互。

- **测试文件数**：5
- **测试用例数**：55

## 运行方式

```bash
# 运行文件与存储相关测试
export $(grep -v '^#' .env | grep -v '^$' | xargs) && mvn test -pl video -Pspringboot \
  -Dtest="com.github.makewheels.video2022.file.*,com.github.makewheels.video2022.oss.*,com.github.makewheels.video2022.utils.OssPathUtilTest" \
  -Dlog4j.configuration=file:///tmp/log4j.properties
```
