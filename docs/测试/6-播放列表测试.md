# 播放列表测试

播放列表模块覆盖播放列表的完整 CRUD 生命周期：创建、更新、删除、恢复、视频添加/移除/排序、详情查询和权限校验。

- **测试文件数**：1
- **测试用例数**：22

## 运行方式

```bash
# 运行播放列表测试
export $(grep -v '^#' .env | grep -v '^$' | xargs) && mvn test -pl video -Pspringboot \
  -Dtest=com.github.makewheels.video2022.playlist.PlaylistServiceTest \
  -Dlog4j.configuration=file:///tmp/log4j.properties
```
