# 基础设施测试

基础设施模块覆盖参数校验服务（CheckService）、Redis 操作、钉钉通知、IP 定位、ID 生成等基础能力，以及冒烟测试和上传场景测试。其中 CheckService 包含 24 个内部测试类，是项目中测试用例最多的文件。

- **测试文件数**：8
- **测试用例数**：115

## 运行方式

```bash
# 运行基础设施相关测试
export $(grep -v '^#' .env | grep -v '^$' | xargs) && mvn test -pl video -Pspringboot \
  -Dtest="com.github.makewheels.video2022.etc.**,com.github.makewheels.video2022.utils.*,com.github.makewheels.video2022.SmokeTest,com.github.makewheels.video2022.scenario.UploadFlowTest" \
  -Dlog4j.configuration=file:///tmp/log4j.properties
```
