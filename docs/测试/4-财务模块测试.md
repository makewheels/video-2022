# 财务模块测试

财务模块覆盖账单生成、钱包管理、交易记录、单价计算和 OSS 访问费用五个核心功能。测试验证了从流量访问到费用计算、账单汇总、钱包扣款的完整计费链路。

- **测试文件数**：5
- **测试用例数**：54

## 运行方式

```bash
# 运行财务模块全部测试
export $(grep -v '^#' .env | grep -v '^$' | xargs) && mvn test -pl video -Pspringboot \
  -Dtest="com.github.makewheels.video2022.finance.*" \
  -Dlog4j.configuration=file:///tmp/log4j.properties
```
