# 视频模块测试

视频模块覆盖视频的创建、更新、详情查询、就绪处理、文件去重链接以及 YouTube 视频导入功能。测试验证了 USER_UPLOAD 和 YOUTUBE 两种视频类型的完整生命周期。

- **测试文件数**：7
- **测试用例数**：74

## 运行方式

```bash
# 运行视频模块全部测试
export $(grep -v '^#' .env | grep -v '^$' | xargs) && mvn test -pl video -Pspringboot \
  -Dtest="com.github.makewheels.video2022.video.*" \
  -Dlog4j.configuration=file:///tmp/log4j.properties
```
