# 封面与转码测试

封面与转码模块覆盖视频封面生成（阿里云截图和 YouTube 缩略图）、转码回调处理、阿里云 MPS 服务调用和云函数转码四个功能。测试验证了从截图/转码任务提交到回调处理的完整流程。

- **测试文件数**：4
- **测试用例数**：37

## 运行方式

```bash
# 运行封面与转码相关测试
export $(grep -v '^#' .env | grep -v '^$' | xargs) && mvn test -pl video -Pspringboot \
  -Dtest="com.github.makewheels.video2022.cover.*,com.github.makewheels.video2022.transcode.*" \
  -Dlog4j.configuration=file:///tmp/log4j.properties
```
