# 部署指南

## 环境要求

- **Java 21** (推荐使用 OpenJDK 21)
- **Maven 3.8+**
- **MongoDB 6.0+** (本地或云端)
- **Redis 7.0+** (本地或云端)
- **阿里云账号** (OSS、MPS、函数计算)

## 1. 本地开发环境搭建

### 1.1 安装依赖

```bash
# 克隆项目
git clone https://github.com/makewheels/video-2022.git
cd video-2022

# 编译项目
mvn clean compile
```

### 1.2 启动 MongoDB

```bash
# macOS (使用 Homebrew)
brew install mongodb-community
brew services start mongodb-community

# 或使用 Docker
docker run -d --name mongodb -p 27017:27017 mongo:6
```

### 1.3 启动 Redis

```bash
# macOS (使用 Homebrew)
brew install redis
brew services start redis

# 或使用 Docker
docker run -d --name redis -p 6379:6379 redis:7
```

### 1.4 配置应用

编辑 `video/src/main/resources/application.properties`：

```properties
# MongoDB 配置
spring.data.mongodb.host=localhost
spring.data.mongodb.port=27017
spring.data.mongodb.database=video-2022
spring.data.mongodb.authentication-database=admin
spring.data.mongodb.username=root
spring.data.mongodb.password=your_password

# Redis 配置
spring.redis.host=localhost
spring.redis.port=6379

# 阿里云 OSS 配置
aliyun.oss.video.accessKeyId=your_access_key_id
aliyun.oss.video.secretKey=your_secret_key
aliyun.oss.video.bucket=your_bucket
aliyun.oss.video.endpoint=oss-cn-beijing.aliyuncs.com

# 阿里云 MPS 配置
aliyun.mps.accessKeyId=your_access_key_id
aliyun.mps.secretKey=your_secret_key

# RSA 私钥文件路径 (用于解密配置文件中的敏感信息)
# macOS: ~/workSpace/~keys/video-2022/privateKey.txt
# Windows: D:/workSpace/~keys/video-2022/privateKey.txt
```

### 1.5 运行应用

```bash
cd video
mvn spring-boot:run
```

应用启动后访问：http://localhost:5022

## 2. 生产环境部署

### 2.1 打包应用

```bash
# 打包 video 模块
mvn clean package -pl video -Pspringboot -DskipTests

# 打包 youtube 模块 (云函数)
mvn clean package -pl youtube -Pyoutube -DskipTests
```

### 2.2 运行 Jar 包

```bash
# 设置 JAVA_HOME (使用 Java 21)
export JAVA_HOME=$(/usr/libexec/java_home -v 21)

# 运行 video 服务
nohup java -jar video/target/video-0.0.1-SNAPSHOT.jar \
  --spring.profiles.active=prod > video.log 2>&1 &

# 查看日志
tail -f video.log
```

### 2.3 配置生产环境

创建 `video/src/main/resources/application-prod.properties`：

```properties
# 生产环境配置
server.port=5022

# MongoDB
spring.data.mongodb.host=your-mongo-host
spring.data.mongodb.port=27017
spring.data.mongodb.database=video-2022
spring.data.mongodb.username=root
spring.data.mongodb.password=your_password

# Redis
spring.redis.host=your-redis-host
spring.redis.port=6379

# 阿里云配置 (建议使用环境变量或配置中心管理敏感信息)
aliyun.oss.video.accessKeyId=${OSS_ACCESS_KEY_ID}
aliyun.oss.video.secretKey=${OSS_SECRET_KEY}
```

## 3. Docker 部署

### 3.1 构建 Docker 镜像

```dockerfile
# Dockerfile
FROM eclipse-temurin:21-jre-alpine
WORKDIR /app
COPY video/target/video-0.0.1-SNAPSHOT.jar app.jar
EXPOSE 5022
ENTRYPOINT ["java", "-jar", "app.jar"]
```

```bash
# 构建镜像
docker build -t video-2022:latest .

# 运行容器
docker run -d \
  -p 5022:5022 \
  -e SPRING_DATA_MONGODB_HOST=mongodb \
  -e SPRING_DATA_MONGODB_USERNAME=root \
  -e SPRING_DATA_MONGODB_PASSWORD=password \
  -e SPRING_REDIS_HOST=redis \
  --name video-2022 \
  video-2022:latest
```

### 3.2 Docker Compose

```yaml
version: '3.8'
services:
  mongodb:
    image: mongo:6
    environment:
      MONGO_INITDB_ROOT_USERNAME: root
      MONGO_INITDB_ROOT_PASSWORD: password
    volumes:
      - mongo-data:/data/db
    ports:
      - "27017:27017"

  redis:
    image: redis:7
    ports:
      - "6379:6379"

  video-app:
    image: video-2022:latest
    environment:
      SPRING_DATA_MONGODB_HOST: mongodb
      SPRING_DATA_MONGODB_USERNAME: root
      SPRING_DATA_MONGODB_PASSWORD: password
      SPRING_REDIS_HOST: redis
    ports:
      - "5022:5022"
    depends_on:
      - mongodb
      - redis

volumes:
  mongo-data:
```

## 6. 常见问题

### 6.1 RSA 密码解密失败

确保私钥文件存在且路径正确：
- macOS: `~/workSpace/~keys/video-2022/privateKey.txt`
- Windows: `D:/workSpace/~keys/video-2022/privateKey.txt`

或者通过环境变量设置：
```bash
export video_2022_privateKey="your_private_key"
```

### 6.2 MongoDB 连接失败

检查 MongoDB 是否启动：
```bash
mongosh --eval "db.adminCommand('ping')"
```

### 6.3 Redis 连接失败

检查 Redis 是否启动：
```bash
redis-cli ping
# 应返回 PONG
```
